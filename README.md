# Playwright 2A — Advanced UI Components

[![Playwright Tests](https://github.com/pyardley/Playwright_2A-Advanced-UI-Components/actions/workflows/playwright.yml/badge.svg)](https://github.com/pyardley/Playwright_2A-Advanced-UI-Components/actions/workflows/playwright.yml)

A Playwright + TypeScript test suite for [UI Testing Playground](http://uitestingplayground.com/), exercising advanced UI component interactions (dynamic IDs, dynamic content, overlapping elements, etc.) with the Page Object Model.

## What this demonstrates

- Page Object Model with a shared `BasePage` and per-page classes exposing behaviour methods (no assertions inside page objects).
- Fixture-driven page object wiring (`fixtures/fixtures.ts`) instead of ad-hoc construction in tests.
- Accessibility-first locators (`getByRole`, `getByLabel`, `getByText`) over brittle CSS/structural selectors.
- Tag-based suite tiers (`@smoke`, `@e2e`) run separately in CI, each with its own retry/report configuration.
- Cross-browser coverage (Chromium, Firefox, WebKit).
- Claude Code agents (`playwright-test-planner`, `playwright-test-generator`, `playwright-test-healer`) for plan → spec → repair workflows.

## Project structure

```
pages/       Page objects — extend BasePage, expose a `path` + behaviour methods
components/  Shared multi-element widgets (nav bars, modals) composed into page objects
fixtures/    Test fixtures wiring page objects into `test`
support/     Shared step sequences and test-data helpers spanning multiple page objects
tests/       Spec files
specs/       Markdown test plans generated by the planner agent
```

## Setup

```bash
npm install
npx playwright install --with-deps
cp .env.example .env
```

## Running tests

```bash
npm test               # full suite, all browsers
npm run test:smoke     # @smoke-tagged tests only
npm run test:e2e       # @e2e-tagged tests only
npm run test:ui        # Playwright UI mode
npm run test:headed    # headed browsers
npm run report         # open the last HTML report
```

## Other scripts

npm run lint       # eslint
